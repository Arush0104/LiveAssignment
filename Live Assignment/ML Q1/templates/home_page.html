Mushroom Classification

Using Machine Learning

How to use this Model Effectively

This project predicts whether a mushroom is poisonous or not based on various features.

To use this, type in the respective observations in the input field
Example:
Lets say I found a mushroom with a cap the shape of a bell.
My input in the cap-shape field should be "b". "b" indicating Bell.
You can describe more characteristcs and type it in from the guide below

Note: Ensure that the information provided is accurate for correct predictions.

You can find the model here.

About the Machine Learning Model

We employed Logistic Regression, suited for categorical data, achieving a 99.8% training accuracy. This indicates a strong fit with our dataset. Further testing with new data is essential for broader validation.
